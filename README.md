## How to Use This Repo

Each design entry includes:
- **Preview Image** – A screenshot of the website design
- **Description** – A short summary of the style, layout, and use case
- **Live Link** – A link to explore the design in the browser

## Contributing

To add a new design, open a pull request with:

1. A screenshot placed in the `images/` folder (e.g., `images/my-design.png`)
2. A new entry in this README following the template below:

```markdown
### Design Name

![Design Name Preview](images/my-design.png)

**Description:** A brief description of the design — its style, color palette, intended use case, and any notable features.

**Live Link:** [Explore Design](https://your-design-url.com)

---
```
